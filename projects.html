Software Development Projects

  1. Virtual DVD Library:

    Developed a CRUD console application using the MVC design pattern and dependency injection to track the contents of a DVD library allowing to add, delete and edit a DVD in the library.

    Technologies Used: Java, MVC Architecture, CSV file input/output, junit testing, Spring DI.

  2. Flooring Company Sales Order System:

    Programmed a CRUD console application that tracks customer orders for a flooring company, allowing a customer to add, delete or edit an order.

    Technologies Used: Java, MVC Architecture, CSV file input/output, junit testing, Spring DI.

  3. Burger Shop Sales Order System:

    Created a user interface for a restaurant called Burger Shop that allows customers to place custom orders, to cancel orders and displays orders created.

    Technologies Used: Python, CSV file input/output, object-oriented programming

  4. Hotel Database:

    Converted a hotel management system from using Excel spreadsheets to a database to keep track of their room reservations, invoices, customers, promotions and room charges.

    Technologies Used: SQL, MySQL

  5. Food Shopping List:

    Developed a simple food shopping list system that allows the user to view the shopping list, add, remove an item on the shopping list, clear the shopping list, and check to see if an item is on the shopping list and how many items are on the shopping list

    Technologies Used: Python, CSV file input/output
